# Cover Page Generator

A modern and professional web-based application to generate academic and project cover pages.

## Features
- Clean & responsive UI
- Professional academic layout
- Live preview
- Easy customization

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

## Usage
1. Open `index.html`
2. Fill in the details
3. Generate your cover page

## Future Enhancements
- PDF export
